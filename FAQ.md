# FAQ

## What is podcast-show-notes?
Show notes generator from audio transcripts. Timestamps, key points, links.

## How do I install it?
```bash
git clone https://github.com/josezuma/podcast-show-notes.git && cd podcast-show-notes
```

## Is this compatible with yc-pitch-deck-max?
Yes. podcast-show-notes integrates directly with yc-pitch-deck-max.

## Do I need any external dependencies?
No. All scripts use only Python stdlib.
